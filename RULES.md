# 开发规则

## 禁止
* 双语言
* 长名字 (名字≤4字)
* 废按钮
* 假数据
* 空页面
* 满屏文字
* 装饰组件
* 新闻页面

## 规则
* 名字≤4字
* 一个功能一页
* 一个按钮一件事
* 所有按钮可点击
* 所有数据真实
* 所有弹窗可提交
* 所有表格可搜索
* 所有列表可分页
* 所有状态可切换
* 所有模块有权限

## 要求
* Shopify风格
* Card布局
* 3D悬浮
* Drawer交互
* 深色侧栏
* 白色卡片
* 大留白
* 少文字

## 禁止
* 满屏说明
* 没有UI
* 假按钮
* 静态页面
* Demo数据
* 花里胡哨

## 目录与菜单 (menu.json)
* 控制
  - 首页
  - 数据
* 平台
  - 店铺
  - 用户
  - 套餐
  - 订阅
* 运营
  - 财务
  - 工单
  - 公告
* 资源
  - 应用
  - 模板
  - 文件
* 系统
  - API
  - 权限
  - 日志
  - 设置

## 页面统一结构
* 顶部：标题、搜索、筛选、新建
* 主体：Card、Table
* 右侧：Drawer
* 底部：分页
